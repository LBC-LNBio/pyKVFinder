pyKVFinder.read_vdwÔÉÅ

pyKVFinder.read_vdw(fn: str | Path | None = None) → Dict[str, Dict[str, float]][source]ÔÉÅ

Reads van der Waals radii from .dat file.

Parameters:

fn (Optional[Union[str, pathlib.Path]], optional) ‚Äì A path to a van der Waals radii file, by default None. If None, apply the built-in van der Waals radii file: vdw.dat.

Returns:

vdw ‚Äì A dictionary containing radii values.

Return type:

Dict[str, Dict[str, float]]

Raises:

  • TypeError ‚Äì fn must be a string or a pathlib.Path.

  • ValueError ‚Äì A line in vdw has incorrect format. The values must be double tab-separated.

  • ValueError ‚Äì A line in vdw has an incorrect radius type for an atom.

Note

The van der Waals radii file defines the radius values for each atom by residue and when not defined, it uses a generic value based on the atom type (see van der Waals file template). The package contains a built-in van der Waals radii file: vdw.dat.

See also

read_pdb, read_xyz, Molecule

The van der Waals radii can be define by:

  • creating a Python dictionary:

>>> vdw = {'GEN': {'C': 1.66, 'CA': 2.0, 'N': 1.97, 'O': 1.69, 'H': 0.91}}

  • specifying a .dat file following template of van der Waals radii file.

>>> with open('vdw.dat', 'w') as f:
...     f.write('>GEN\nC\t\t1.66\nCA\t\t2.00\nN\t\t1.97\nO\t\t1.69\nH\t\t0.91\n')
>>> vdw = read_vdw('vdw.dat')
>>> vdw
