pyKVFinder.export(fn: Union[str, pathlib.Path], cavities: numpy.ndarray, surface: Optional[numpy.ndarray], vertices: Union[numpy.ndarray, List[List[float]]], step: Union[float, int] = 0.6, B: Optional[numpy.ndarray] = None, output_hydropathy: Union[str, pathlib.Path] = 'hydropathy.pdb', scales: Optional[numpy.ndarray] = None, selection: Optional[Union[List[int], List[str]]] = None, nthreads: Optional[int] = None, append: bool = False, model: int = 0) → None[source]Â¶

Exports cavitiy (H) and surface (HA) points to PDB-formatted file with a variable (B; optional) in B-factor column, and hydropathy to PDB-formatted file in B-factor column at surface points (HA).

Parameters

  • fn (Union[str, pathlib.Path]) â A path to PDB file for writing cavities.

  • cavities (numpy.ndarray) â

    Cavity points in the 3D grid (cavities[nx][ny][nz]). Cavities array has integer labels in each position, that are:

    • -1: bulk points;

    • 0: biomolecule points;

    • 1: empty space points;

    • >=2: cavity points.

    The empty space points are regions that do not meet the chosen volume cutoff to be considered a cavity.

  • surface (numpy.ndarray, optional) â

    Surface points in the 3D grid (surface[nx][ny][nz]). If None, surface is a numpy.zeros array with same shape of cavities. Surface array has integer labels in each position, that are:

    • -1: bulk points;

    • 0: biomolecule or empty space points;

    • >=2: surface points.

    The empty space points are regions that do not meet the chosen volume cutoff to be considered a cavity.

  • vertices (Union[numpy.ndarray, List[List[float]]]) â A numpy.ndarray or a list with xyz vertices coordinates (origin, X-axis, Y-axis, Z-axis).

  • step (Union[float, int], optional) â Grid spacing (A), by default 0.6.

  • B (Union[numpy.ndarray, None], optional) â A numpy.ndarray with values to be mapped on B-factor column in cavity points (B[nx][ny][nz]), by default None.

  • output_hydropathy (Union[str, pathlib.Path], optional) â A path to hydropathy PDB file (surface points mapped with a hydrophobicity scale), by default hydropathy.pdb.

  • scales (Union[numpy.ndarray, None], optional) â A numpy.ndarray with hydrophobicity scale values to be mapped on B-factor column in surface points (scales[nx][ny][nz]), by default None.

  • selection (Union[List[int], List[str]], optional) â A list of integer labels or a list of cavity names to be selectedA list of integer labels of each cavity to be selected, by default None.

  • nthreads (int, optional) â Number of threads, by default None. If None, the number of threads is os.cpu_count() - 1.

  • append (bool, optional) â Whether to append cavities to the PDB file, by default False.

  • model (int, optional) â Model number, by default 0.

Raises

  • TypeError â cavities must be a numpy.ndarray.

  • ValueError â cavities has the incorrect shape. It must be (nx, ny, nz).

  • TypeError â surface must be a numpy.ndarray.

  • ValueError â surface has the incorrect shape. It must be (nx, ny, nz).

  • TypeError â vertices must be a list or a numpy.ndarray.

  • ValueError â vertices has incorrect shape. It must be (4, 3).

  • TypeError â step must be a positive real number.

  • ValueError â step must be a positive real number.

  • TypeError â B must be a numpy.ndarray.

  • ValueError â B has the incorrect shape. It must be (nx, ny, nz).

  • TypeError â scales must be a numpy.ndarray.

  • ValueError â scales has the incorrect shape. It must be (nx, ny, nz).

  • TypeError â selection must be a list of strings (cavity names) or integers (cavity labels).

  • ValueError â Invalid selection: {selection}.

  • TypeError â nthreads must be a positive integer.

  • ValueError â nthreads must be a positive integer.

  • TypeError â append must be a boolean.

  • TypeError â model must be a integer.

  • TypeError â fn must be a string or pathlib.Path.

  • TypeError â output_hydropathy must be a string.

  • Exception â User must define surface when not defining cavities.

Note

The cavity nomenclature is based on the integer label. The cavity marked with 2, the first integer corresponding to a cavity, is KAA, the cavity marked with 3 is KAB, the cavity marked with 4 is KAC and so on.

Example

With the cavity and surface points identified and depth and hydrophobicity scale mapped in the 3D grid, we can:

  • Export cavity points

>>> from pyKVFinder import export
>>> export('cavity_wo_surface.pdb', cavities, None, vertices)

  • Export cavity and surface points

>>> export('cavities.pdb', cavities, surface, vertices)

  • Export cavity and surface points with depth mapped on them

>>> export('cavities_with_depth.pdb', cavities, surface, vertices, B=depths)

  • Export surface points with hydrophobicity_scale mapped on them

>>> export(None, None, surface, vertices, output_hydropathy='hydropathy.pdb', scales=scales)

  • Export all

>>> export('cavities.pdb', cavities, surface, vertices, B=depths, output_hydropathy='hydropathy.pdb', scales=scales)
