Reads PDB file into numpy.ndarrays.

Parameters

  • fn (Union[str, pathlib.Path]) â A path to PDB file.

  • vdw (Dict[str, Dict[str, float]) â A dictionary with radii values, by default pyKVFinder.read_vdw().

Returns

  • atominfo (numpy.ndarray) â A numpy array with atomic information (residue number, chain, residue name, atom name) for each atom.

  • xyzr (numpy.ndarray) â A numpy.ndarray with xyz atomic coordinates and radii for each atom.

Raises

TypeError â fn must be a string or a pathlib.Path.

Note

The van der Waals radii file defines the radius values for each atom by residue and when not defined, it uses a generic value based on the atom type. The function by default loads the built-in van der Waals radii file: vdw.dat.

  • van der Waals file template

Example

With the vdW radii dictionary loaded with pyKVFinder.read_vdw, we can read a target PDB file into Numpy arrays (atomic information and atomic coordinates):

>>> import os
>>> from pyKVFinder import read_pdb
>>> pdb = os.path.join(os.path.dirname(pyKVFinder.__file__), 'data', 'tests', '1FMO.pdb')
>>> atominfo, xyzr = read_pdb(pdb)
>>> atominfo
array([['13_E_GLU', 'N'],
   ['13_E_GLU', 'CA'],
   ['13_E_GLU', 'C'],
   ...,
   ['350_E_PHE', 'CE2'],
   ['350_E_PHE', 'CZ'],
   ['350_E_PHE', 'OXT']], dtype='<U9')
>>> xyzr
array([[ -6.693, -15.642, -14.858,   1.824],
   [ -6.73 , -14.62 , -15.897,   1.908],
   [ -7.49 , -13.357, -15.508,   1.908],
   ...,
   [  7.216,  18.878,  -9.885,   1.908],
   [  7.735,  17.624,  -9.558,   1.908],
   [  5.767,  19.234, -13.442,   1.69 ]])

Warning

The function takes the built-in dictionary when the vdw argument is not specified. If you wish to use a custom van der Waals radii file, you must read it with pyKVFinder.read_vdw as shown earlier and pass it as pyKVFinder.read_pdb(pdb, vdw=vdw).
