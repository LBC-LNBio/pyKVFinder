pyKVFinder.constitutionalÔÉÅ

pyKVFinder.constitutional(cavities: ndarray, atomic: ndarray | List[List[str | float | int]], vertices: ndarray | List[List[float]], step: float | int = 0.6, probe_in: float | int = 1.4, ignore_backbone: bool = False, selection: List[int] | List[str] | None = None, nthreads: int | None = None, verbose: bool = False) → Dict[str, List[List[str]]][source]ÔÉÅ

Constitutional characterization (interface residues) of the detected cavities.

Parameters:

  • cavities (numpy.ndarray) ‚Äì

    Cavity points in the 3D grid (cavities[nx][ny][nz]). Cavities array has integer labels in each position, that are:

    • -1: bulk points;

    • 0: biomolecule points;

    • 1: empty space points;

    • >=2: cavity points.

    The empty space points are regions that do not meet the chosen volume cutoff to be considered a cavity.

  • atomic (Union[numpy.ndarray, List[List[Union[str, float, int]]]]) ‚Äì A numpy array with atomic data (residue number, chain, residue name, atom name, xyz coordinates and radius) for each atom.

  • vertices (Union[numpy.ndarray, List[List[float]]]) ‚Äì A numpy.ndarray or a list with xyz vertices coordinates (origin, X-axis, Y-axis, Z-axis).

  • step (Union[float, int], optional) ‚Äì Grid spacing (A), by default 0.6.

  • probe_in (Union[float, int], optional) ‚Äì Probe In size (A), by default 1.4.

  • ignore_backbone (bool, optional) ‚Äì Whether to ignore backbone atoms (C, CA, N, O) when defining interface residues, by default False.

  • selection (Union[List[int], List[str]], optional) ‚Äì A list of integer labels or a list of cavity names to be selectedA list of integer labels of each cavity to be selected, by default None.

  • nthreads (int, optional) ‚Äì Number of threads, by default None. If None, the number of threads is os.cpu_count() - 1.

  • verbose (bool, optional) ‚Äì Print extra information to standard output, by default False.

Returns:

residues ‚Äì A dictionary with a list of interface residues for each detected cavity.

Return type:

Dict[str, List[List[str]]]

Raises:

  • TypeError ‚Äì cavities must be a numpy.ndarray.

  • ValueError ‚Äì cavities has the incorrect shape. It must be (nx, ny, nz).

  • TypeError ‚Äì atomic must be a list or a numpy.ndarray.

  • ValueError ‚Äì atomic has incorrect shape. It must be (n, 8).

  • TypeError ‚Äì vertices must be a list or a numpy.ndarray.

  • ValueError ‚Äì vertices has incorrect shape. It must be (4, 3).

  • TypeError ‚Äì step must be a positive real number.

  • ValueError ‚Äì step must be a positive real number.

  • TypeError ‚Äì probe_in must be a non-negative real number.

  • ValueError ‚Äì probe_in must be a non-negative real number.

  • TypeError ‚Äì ignore_backbone must be a boolean.

  • TypeError ‚Äì selection must be a list of strings (cavity names) or integers (cavity labels).

  • ValueError ‚Äì Invalid selection: {selection}.

  • TypeError ‚Äì nthreads must be a positive integer.

  • ValueError ‚Äì nthreads must be a positive integer.

  • TypeError ‚Äì verbose must be a boolean.

Note

The cavity nomenclature is based on the integer label. The cavity marked with 2, the first integer corresponding to a cavity, is KAA, the cavity marked with 3 is KAB, the cavity marked with 4 is KAC and so on.

Note

The classes of residues are:

  • Aliphatic apolar (R1): Alanine, Glycine, Isoleucine, Leucine, Methionine, Valine.

  • Aromatic (R2): Phenylalanine, Tryptophan, Tyrosine.

  • Polar Uncharged (R3): Asparagine, Cysteine, Glutamine, Proline, Serine, Threonine.

  • Negatively charged (R4): Aspartate, Glutamate.

  • Positively charged (R5): Arginine, Histidine, Lysine.

  • Non-standard (RX): Non-standard residues.

See also

read_pdb, read_xyz, detect, calculate_frequencies, write_results
