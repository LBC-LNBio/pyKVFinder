pyKVFinder.depthÔÉÅ

pyKVFinder.depth(cavities: ndarray, step: float | int = 0.6, selection: List[int] | List[str] | None = None, nthreads: int | None = None, verbose: bool = False) → Tuple[ndarray, Dict[str, float], Dict[str, float]][source]ÔÉÅ

Characterization of the depth of the detected cavities, including depth per cavity point and maximum and average depths of detected cavities.

Parameters:

  • cavities (numpy.ndarray) ‚Äì

    Cavity points in the 3D grid (cavities[nx][ny][nz]). Cavities array has integer labels in each position, that are:

    • -1: bulk points;

    • 0: biomolecule points;

    • 1: empty space points;

    • >=2: cavity points.

    The empty space points are regions that do not meet the chosen volume cutoff to be considered a cavity.

  • step (Union[float, int], optional) ‚Äì Grid spacing (A).

  • selection (Union[List[int], List[str]], optional) ‚Äì A list of integer labels or a list of cavity names to be selected, by default None.

  • nthreads (int, optional) ‚Äì Number of threads, by default None. If None, the number of threads is os.cpu_count() - 1.

  • verbose (bool, optional) ‚Äì Print extra information to standard output, by default False.

Returns:

  • depths (numpy.ndarray) ‚Äì A numpy.ndarray with depth of cavity points (depth[nx][ny][nz]).

  • max_depth (Dict[str, float]) ‚Äì A dictionary with maximum depth of each detected cavity.

  • avg_depth (Dict[str, float]) ‚Äì A dictionary with average depth of each detected cavity.

Raises:

  • TypeError ‚Äì cavities must be a numpy.ndarray.

  • ValueError ‚Äì cavities has the incorrect shape. It must be (nx, ny, nz).

  • TypeError ‚Äì step must be a positive real number.

  • ValueError ‚Äì step must be a positive real number.

  • TypeError ‚Äì selection must be a list of strings (cavity names) or integers (cavity labels).

  • ValueError ‚Äì Invalid selection: {selection}.

  • TypeError ‚Äì nthreads must be a positive integer.

  • ValueError ‚Äì nthreads must be a positive integer.

  • TypeError ‚Äì verbose must be a boolean.

Note

The cavity nomenclature is based on the integer label. The cavity marked with 2, the first integer corresponding to a cavity, is KAA, the cavity marked with 3 is KAB, the cavity marked with 4 is KAC and so on.

See also

detect, export, write_results

Example

With the cavity points identified with detect, we can perform a depth characterization, that includes maximum depth, average depth and defining depth of cavity points:

>>> from pyKVFinder import depth
>>> depths, max_depth, avg_depth = depth(cavities)

        [0., 0., 0., ..., 0., 0., 0.],
