Issues
======

pyKVFinder package uses of GitHub for project management. Please help us by reporting any problems or requests.

On `Issues <https://github.com/LBC-LNBio/pyKVFinder/issues>`_ page, you can file a:

.. toctree::
    
    Bug Report <https://github.com/LBC-LNBio/pyKVFinder/issues/new?assignees=&labels=&template=bug_report.md&title=>
    Request for a new feature <https://github.com/LBC-LNBio/pyKVFinder/issues/new?assignees=&labels=&template=feature_request.md&title=>

Only make sure the bug or request has not already been reported. Click on “Search” and enter some keywords to search.

Thank you for helping us improve pyKVFinder!
